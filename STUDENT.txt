# UStory App

## Pengantar

Proyek ini dibuat sebagai bagian dari kelas [Menjadi Front-End Web Developer Expert](https://www.dicoding.com/academies/219).  
UStory adalah aplikasi berbasis Progressive Web App (PWA) yang memungkinkan pengguna membagikan cerita mereka melalui foto, deskripsi, dan lokasi pada peta interaktif.  
Aplikasi ini mendukung mode offline, push notification, instalasi ke homescreen, serta penyimpanan data lokal menggunakan IndexedDB.

## Deskripsi

UStory dikembangkan dengan konsep SPA (Single Page Application) menggunakan JavaScript modular dan Webpack.  
Fitur-fitur utama meliputi:
- Autentikasi (Login & Register)
- Tambah Story dengan foto & lokasi (geolocation)
- Peta Interaktif menggunakan Leaflet
- Offline Mode (cache app shell & data dinamis)
- Push Notification dengan VAPID keys
- IndexedDB untuk penyimpanan lokal (create, read, delete)
- Filter, Search, dan Sort Story
- Mode Gelap (Dark Mode)
- PWA Installable + Manifest lengkap

## Prasyarat

- Node.js (versi terbaru disarankan)
- npm

## Instalasi

Pasang dependensi:
    ```bash
    npm install
Jalankan dalam mode development:
    ```bash
    npm run start-dev
Bangun untuk production:
    ```bash
    npm run build
Jalankan hasil build:
    ```bash
    npm run serve
